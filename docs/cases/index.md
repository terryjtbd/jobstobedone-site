# 案例研究

## 案例总览

这里记录我们运用 JTBD 方法论框架帮助客户解决真实商业问题的案例。

每个案例都遵循统一的分析结构：

1. **背景**：客户是谁，面临什么挑战
2. **JTBD 洞察**：我们发现了什么待办任务
3. **战略设计**：基于洞察的战略选择
4. **执行落地**：关键举措和成果
5. **关键启示**：可迁移的经验
